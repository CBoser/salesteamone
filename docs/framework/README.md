## Framework Principles

### Sustainability Over Speed
- 30-minute focused sessions
- 60-90 minutes per day max
- Wednesday protected for caregiving
- Sustainable long-term pace

### Quality Over Quantity
- 70%+ test coverage required
- All tests passing before sprint close
- No TypeScript errors tolerated
- Comprehensive validation

### Documentation Over Memory
- Daily progress logging
- Decision documentation
- Lessons learned tracking
- User-facing changelogs

### Predictability Over Uncertainty
- Weekly velocity tracking
- Data-driven capacity planning
- Conservative task estimates
- Adaptive sprint planning
