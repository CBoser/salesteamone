# Comprehensive Project Assessment Questionnaire
## Strategic Project Planning

**Purpose**: Capture complete project requirements and constraints

**Time Required**: 45-60 minutes

**Frequency**: At project inception

```
Help me plan a new project by asking strategic questions.

## PHASE 1: ESSENTIAL CONTEXT (10 questions)

Ask these first:
1. What problem does this solve?
2. Who are the users?
3. What's the timeline?
4. What's the budget?
5. What are must-have features?
6. What are deal-breaker constraints?
7. What does success look like?
8. What existing systems must integrate?
9. Who are key stakeholders?
10. What's the biggest risk?

## PHASE 2: TARGETED DEEP DIVE

Based on Phase 1, ask follow-ups:
- If B2B: Enterprise requirements
- If high-scale: Performance requirements
- If greenfield: Technology selection
- If migration: Data migration strategy

## OUTPUT: PROJECT BRIEF

Generate:
1. **Problem Statement** (1 paragraph)
2. **Success Criteria** (measurable outcomes)
3. **Key Constraints** (timeline, budget, technical)
4. **Primary Users** (personas)
5. **Technical Approach** (recommended architecture)
6. **Implementation Roadmap** (phases with estimates)
7. **Risk Mitigation** (top 3 risks + plans)
8. **Open Questions** (needs clarification)
```
