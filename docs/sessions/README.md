## Purpose

Session notes capture:
- Work accomplished during each development session
- Decisions made and rationale
- Blockers encountered and resolutions
- Next steps and action items

Weekly wrap-ups provide:
- Summary of week's progress
- Velocity and completion metrics
- Lessons learned
- Planning for next week

## Organization

Files are organized chronologically by date (YYYY-MM-DD format).

For current active session tracking, use the framework tools:
```bash
python tools/log_time.py start
python tools/log_time.py end "What you accomplished"
python tools/log_time.py summary
```
